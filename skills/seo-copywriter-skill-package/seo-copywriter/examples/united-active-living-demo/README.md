# United Active Living — Senior Living Content Demo

A finished, publish-ready **SEO + GEO content sample** prepared for **United Active Living** (six senior-living communities in Calgary, Alberta). It demonstrates the kind of source-grounded, E-E-A-T-ready, AI-answer-optimized content the SEO Copywriter workflow produces — dressed in final design so it can be shown to the client.

Built by mimicking the Whispering Willows content demo: same structure and quality bar, re-themed to United Active Living's navy + gold brand, and re-grounded entirely in **Canadian / Alberta** facts.

## What's here

A small static site (no build step — plain HTML/CSS/JS):

- **`index.html`** — strategy hub: the topic cluster across the awareness → consideration → decision funnel.
- **`senior-living-calgary.html`** — *Senior Living in Calgary: A Family's Guide* (pillar).
- **`independent-assisted-memory-care-alberta.html`** — *Independent vs Assisted vs Memory Care in Alberta* (comparison).
- **`cost-of-senior-living-calgary.html`** — *The Cost of Senior Living in Calgary (2026)* (the flagship cornerstone).
- **`signs-its-time.html`** — *Signs It's Time for Assisted Living or Memory Care* (decision / YMYL).
- **`choosing-senior-living-calgary.html`** — *Choosing a Community: 12 Questions to Ask on a Tour* (decision).
- **`faq.html`** — sectioned FAQ with `FAQPage` schema.
- **`checklist.html`** — a free, printable tour checklist.
- **`styles.css`** — the warm navy-and-gold design system.
- **`app.js`**, **`sitemap.xml`**, **`robots.txt`**, **`images/`** — supporting files.

## How it reflects the quality bar

- **No fabricated statistics.** Every figure is cited to a named authority — Statistics Canada, the Alzheimer Society of Canada, the Government of Alberta (alberta.ca), Alberta Health Services, and United Active Living's own published pricing.
- **Built for AI answers (GEO).** Structured headings, comparison tables, and `FAQPage` / `Article` / `BreadcrumbList` JSON-LD so answer engines can lift a clean, quotable passage.
- **E-E-A-T ready.** A named, accountable reviewer byline on every page. **For real publication, swap in a named credentialed reviewer** (e.g., a Director of Care / RN) — the trust signal the live blog is missing today.
- **Local SEO.** `SeniorLivingResidence` schema, NAP, and Calgary / Alberta targeting throughout.
- **YMYL-safe & Canadian.** Educational framing, medical/financial disclaimers, Alberta-correct funding (Type A/B continuing care, accommodation charges, the Alberta Seniors Benefit) — **no US Medicare/Medicaid**.

## Deploy

Static site — Vercel auto-detects it (framework preset **Other**, no build command, output directory **`.`**). Import the repo at [vercel.com/new](https://vercel.com/new) and deploy, or `vercel deploy`.
