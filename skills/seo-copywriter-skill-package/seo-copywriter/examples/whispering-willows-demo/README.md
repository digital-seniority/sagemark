# Whispering Willows — Memory Care Content Demo

A finished, publish-ready **SEO + GEO content sample** prepared for **Whispering Willows Memory Care** (Mount Vernon, WA). It demonstrates the kind of source-grounded, E-E-A-T-ready, AI-answer-optimized content the SEO Copywriter engine produces — dressed in final design so it can be shown to the client.

## What's here

A small static site (no build step — plain HTML/CSS/JS):

- **`index.html`** — strategy hub: the three-article plan across the awareness → consideration → decision funnel.
- **`article-early-signs.html`** — *The 10 Early Signs of Dementia in an Aging Parent* (awareness).
- **`article-cost.html`** — *How Much Does Memory Care Cost in Washington? A 2026 Family Guide* (consideration).
- **`article-choosing.html`** — *Choosing Memory Care in Mount Vernon & the Skagit Valley: 12 Questions to Ask on a Tour* (decision).
- **`styles.css`** — the warm, "feels like home" design system.
- **`images/`** — bundled photography (Pexels, free for commercial use).

## How it reflects the engine's quality bar

- **No fabricated statistics.** Every figure is cited to a named authority — the Alzheimer's Association, the National Institute on Aging, Medicare.gov, and current cost data (A Place for Mom / Caring.com).
- **Built for AI answers (GEO).** Structured headings, comparison tables, and `FAQPage` JSON-LD so answer engines can lift a clean, quotable passage.
- **E-E-A-T ready.** A named, accountable reviewer byline on every page (the trust signal the live blog is missing today). For real publication, swap in your named clinical reviewer (e.g., Director of Nursing, RN).
- **Local SEO.** `LocalBusiness` schema, NAP, and Mount Vernon / Skagit Valley targeting throughout.
- **YMYL-safe.** Educational framing, medical/financial disclaimers, and an accurate treatment of what Medicare/Medicaid do and don't cover.

## Deploy

Static site — Vercel auto-detects it (framework preset **Other**, no build command, output directory **`.`**). Import the repo at [vercel.com/new](https://vercel.com/new) and deploy.
